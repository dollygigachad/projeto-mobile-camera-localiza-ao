Como executar o projeto primeiro voce instala as dependencias com npm install no terminal.
Para iniciar o projeto voce usa o comando no terminal npx expo start.
Depois de ter feito isso falta somente escanear o QR Code com o Expo Go no celular e ira aparecer o projeto.

```bash
# Instalar dependências
npm install

# Iniciar projeto
npx expo start 

# Escanear QR Code com Expo Go no celular
```

##  Comandos Úteis
```bash
npm install              # Instalar dependências
npx expo start --tunnel # Iniciar em tunnel
```
